# **IOTBTECH WAREHOUSES**

### Introduction
IOTBTECH WAREHOUSES is a company that owns many warehouses in different places. I was given their transactional purchase dataset to make an invoice out of it. The dataset contains transactional purchase records for agricultural products across multiple warehouses. It captures individual-level buying activity from January to June 2024 and it is structured for sales, inventory and customer analysis. 
  This dataset is suitable for analyzing sales trends by product,warehouse performance, customer purchasing patterns and revenue by category for an agri-retail business.
